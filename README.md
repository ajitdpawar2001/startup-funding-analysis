# 🚀 Startup Funding Analysis Dashboard

An interactive web application built using Streamlit to analyze funding trends in the Indian startup ecosystem.

---

## 📊 Features
- Overall funding analysis
- Investor deep-dive
- Startup-level insights
- Month-on-Month trends visualization

---

## 🛠 Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib

---

## 📂 Dataset
- Cleaned dataset of startup funding (2015–2020)

---

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run STARTUP.py
